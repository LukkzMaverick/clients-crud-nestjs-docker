# clients-crud-nestjs-docker

## Get Started

```bash
git clone git@github.com:LukkzMaverick/clients-crud-nestjs-docker.git
cd clients-crud-nestjs-docker
docker-compose up -d
docker exec -it node bash
cd clients-api
npm i
```

## Running the app

```bash
npm run start:dev
```

## Test

```bash
# unit tests
npm run test

# test coverage
npm run test:cov
```
